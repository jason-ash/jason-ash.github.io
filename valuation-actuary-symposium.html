Valuation Actuary Symposium

I will be presenting along with my former colleague, Zachary Brown, at the Valuation Actuary Symposium later this month in Washington, D.C. Our topic is "Improving Actuarial Communication". Zack's portion focuses on improving presentation skills, and I will talk about improving visual communication.

Improving Actuarial Communication: 2018 Valuation Actuary Symposium

Given that I am presenting on new visual communication technology, I thought it would be a good opportunity to explore some tools I hadn't used before. I use jupyter notebooks on a daily basis to combine code, visuals, and markdown text all in one place. I wondered if there was a tool that could help me translate a jupyter notebook into a formal presentation. Then I discovered RISE, a code package that allows you to convert jupyter notebooks into amazing slideshows using reveal.js behind the scenes. The final result is a presentation that seamlessly integrates code, visuals, and text in a sleek format that I can host online. Therefore, in addition to the presentation content, the medium itself is an exploration of new visual communication technology. Anyone familiar with jupyter notebooks (I hope this includes many actuaries) can quickly start to explore this new presentation format as well.

Please enjoy my presentation, and reach out to me at jason@ashanalytics.com if I can help you by answering any questions!
